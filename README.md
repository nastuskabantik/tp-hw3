# Домашнее задание №3: Генерация и анализ данных (Танцевальные направления)

## Как открыть HTML-отчёт в браузере (через GitHub Codespaces):

1. Выполните сборку и запуск контейнеров для создания отчета:
   `./run.sh build_generator`
   `./run.sh run_generator`
   `./run.sh build_reporter`
   `./run.sh run_reporter`
2. В папке `data/` появится файл `report.html`.
3. Запустите веб-сервер с помощью команды: 
   `./run.sh report_server`
   (Она поднимет стандартный образ nginx, пробросив порт 8080 на хост и смонтировав папку `data` как источник статики `ro` — read-only).
4. Перейдите на вкладку **Ports** (Порты) в нижней панели VS Code / Codespaces.
5. Найдите проброшенный порт **8080** и нажмите иконку **Open in Browser** (Открыть в браузере).
6. В адресной строке открывшейся вкладки добавьте к ссылке `/report.html`.
7. Для остановки сервера выполните в терминале `docker stop hw_server`.
